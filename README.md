A collaborative project management web application built with HTML, CSS, JavaScript, Git, and Firebase.

## Description
**Join** is a web-based project management tool developed together with other participants from a professional web development training program.

The application helps teams visualize tasks, their current status, and assigned responsibilities in a clear and structured way.  
Users can create, update, and organize tasks to keep track of workflows and team progress efficiently.
This project focuses on real-world web development, teamwork, and the use of modern web technologies.

## Features
- Create and manage tasks
- Assign tasks to team members
- Visualize task status and progress
- User-friendly interface
- Real-time data storage with Firebase
- Collaborative development using Git

## Built With
- HTML5
- CSS3
- JavaScript (ES6)
- Git
- Firebase

## How to Run
1. Clone the repository
2. Open `index.html` in your browser  

## Learning Goals
